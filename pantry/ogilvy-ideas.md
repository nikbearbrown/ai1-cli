# Copywriting principles → seed material for the copywriting book

*Source (provenance only): the "Ogilvy Copywriting Coach" prompt set (Nik's
two-mode expert tool, shared 2026-06-12 in the Cowork session that replanned
this book). This note harvests its structure as chapter/exercise ideas for
the writing-guide → copywriting conversion (outline Ch 9, "Add What's
Missing"). The full prompt is the canonical source; this is the map, not
the territory.*

**USAGE RULE (per author, 2026-06-12):** the book references these as
*principles of classic direct advertising* — citable, attributable where a
specific claim needs a source — but does **not** use the Ogilvy name as
endorsement, branding, or authority-by-association. The principles must
stand on their own; Chapter 1's research question ("do they still apply in
an AI world?") treats them as hypotheses to test, not doctrine to inherit.

## Principles worth teaching as chapters or sections

- **No copy before the differentiator.** "Never write a word before you can
  state the brand's single differentiator in your own words." → candidate
  opening chapter of the new Part: the brief comes before the copy, exactly
  as the Blueprint comes before the book.
- **Features → outcomes.** "Our platform has 200 integrations" is not a
  benefit. The transformer template: *[Feature] means you can [benefit] so
  that [emotional payoff].* → exercise gold; maps directly onto converting
  writing-guide's "evaluation" and "proposal" chapters.
- **Vague adjectives are interrogation triggers, not copy.** "Innovative,"
  "passionate," "best-in-class" → banned-words discipline; ties to the
  writing guide's existing rhetoric chapters.
- **Earn attention first** — never open body copy with the brand name.
- **Brand voice as intake, not vibes** — the 10-question Brand Voice Intake
  (personality, audience, differentiator, admired brands, off-limits tone,
  fears/dreams, platforms, the one action) + confirmation gate. → this IS a
  chapter: "The Brand Voice File," with the intake as the exercise. Echoes
  the AI+1 pattern: structured intake → confirmed profile → generation.

## What NOT to import

- The persona theatrics (*adjusts spectacles*) — fine for a tool, wrong for a
  textbook register; the *behavioral rules* are the durable part.
- The Medhavy/causal command extensions — product-specific scripts, not
  copywriting pedagogy. (Though the "frame violation flags" pattern — named
  violations with scripted responses — is a teachable editing technique.)
- Hashtag/SEO appendix mechanics in 2026 specifics — date-stamped material;
  if used, isolate in a "current platform constraints" section flagged
  high-aging-risk per Blueprint's aging audit.

## The deeper rhyme (for the introduction, maybe)

Ogilvy's tool and AI+1 share one architecture: intake before generation,
gates before output, pushback as a feature, silent mode as an earned
convenience, and a hard rule that the differentiator (the thesis) must be
statable before anything gets written. The copywriting book can teach
copywriting *and* quietly demonstrate the method that built it.
