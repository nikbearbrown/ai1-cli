# Practice Exercises  →  built as a tool

> **Implemented.** Generated by `build-exercises.py` at the repo root. Run after
> the spine is complete (GATE 6 signed). Worked examples and full annotated
> solutions are produced inline, so this also covers the **Worked Exercises &
> Solutions** edition.

**What it is:** per major concept — an **exploratory problem** (productive
failure), an **annotated worked example**, **2–3 faded examples**, then a
graduated independent set (**Tier 1 Consolidation → Tier 5 Challenge**), plus a
**Cumulative Review** of prior chapters. Integration (Tier 3) appears from the
2nd concept chapter on; Cumulative Review from the 3rd — neither for the
Introduction or Chapter 1. Every item is grounded in the chapter text.

**Inputs:** chapters in `chapters/` (+ a key in `.env`).
**Outputs:** one markdown file per chapter in `editions/exercises/exercises-<stem>.md`.

## Run it

Calibration first (wired to the verified-artifact gate):

```bash
python build-exercises.py               # calibration: first chapter only, then STOP
# review editions/exercises/exercises-00-introduction.md against the 6-point checklist it prints
python scripts/verify.py sign editions/exercises/exercises-00-introduction.md --by <name> --note "calibration ok"
python build-exercises.py --all         # batch the rest (skips any already-verified file)
```

Other modes: `--chapter 04`, `--limit N`, `--all --force`. Config in
`exercises-config.yaml` (`skip_chapters`, `output_dir`, `cumulative_review`).
