# Further Reading Guide

> **Edition stub.** Run only after the spine is complete (GATE 6 signed).
> Expand this into a full prompt when you decide to build it.

**What it is:** An annotated guide to what to read next, organized by goal.

**When to use:** You want to point readers beyond the book without a flat reference dump.

**Inputs:** the finished spine chapters in `chapters/` (and `research/` where relevant).

**Outputs:** `editions/further-reading.md` — annotated, grouped by reader goal.

## Starter prompt

```
Using research/ sources and the field-positioning in vision.md, build an annotated further-reading guide. Group by reader goal (go deeper on X, see the original source, get the opposing view, get the practical version). For each item: full citation, 1-2 sentence annotation on why and for whom, and difficulty. Prefer primary sources.
```
