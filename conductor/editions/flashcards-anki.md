# Flashcards / Anki Deck  →  built as a tool

> **Implemented.** Generated by `build-anki.py` at the repo root. Run after the
> spine is complete (GATE 6 signed).

**What it is:** a spaced-repetition `.apkg` deck — one card per term and concept,
extracted from the chapters, optionally with audio on the card front.

**Inputs:** chapters in `chapters/` (+ `ANTHROPIC_API_KEY` in `.env`; optional
`ELEVENLABS_API_KEY` for audio).
**Outputs:** `output/<book-slug>-anki-<date>.apkg` (+ intermediate cards JSON + log).

## Run it

```bash
python build-anki.py --test            # one chapter, review the cards at the gate
python build-anki.py                   # full deck (text-only by default)
```

Audio is **off by default**; enable with `audio: true` in `anki-config.yaml`
plus `ELEVENLABS_API_KEY`. Test mode caps audio to a few cards. See
`anki-config.yaml` for deck name, voice, and skip settings.
