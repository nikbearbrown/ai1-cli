# conductor/editions/ — Optional Editions

Run these **only after the spine is complete** (all gates in `../../STATUS.md`
signed through GATE 6). Each edition is independent — pick none, some, or all.
Check your choices in `STATUS.md`.

Each file is a **stub**: what the edition is, when to use it, and a starter
prompt block. Expand a stub into a full prompt when you decide to build it.
